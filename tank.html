Tank Battle

A small browser tank game where you move, dodge, and shoot enemy tanks. Clear all enemies to win the round.

Score: 0 Enemies left: 0 Running

Controls

  • Use arrow keys to move the tank.
  • Press the space bar to fire.
  • Destroy all enemy tanks and avoid enemy bullets.

Project Description

This is a simple arcade-style tank battle demo made with HTML, CSS, and JavaScript. It focuses on basic movement, shooting, collision, and win or lose states.
