Tank Battle

A fuller browser tank game with walls, lives, enemy fire, explosions, and level progression.

Score: 0 Enemies left: 0 Lives: 3 Level: 1 Running

Controls

  • Use arrow keys to move the tank.
  • Press the space bar to fire.
  • Bullets can destroy walls. Enemies can also fire back.

Project Description

This version includes destructible walls, player lives, enemy bullets, explosion effects, and multiple levels. Clear each level to continue.

Open Source

This game project is open source. You can view the code or download the file directly.
