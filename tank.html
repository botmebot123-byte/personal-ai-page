Tank Battle

A simple browser tank game. Move with the arrow keys, shoot with the space bar, and defeat all enemy tanks.

Score: 0 Enemies left: 0 Running

Controls

  • Use arrow keys to move the tank.
  • Press the space bar to fire.
  • Destroy all enemy tanks and avoid direct contact.

Project Description

This is a rebuilt stable version of the tank game made with HTML, CSS, and JavaScript. It focuses on smooth movement, simple shooting, and clear win or lose states.

Open Source

This game project is open source. You can view the code or download the file directly.
