Snake Game

A simple browser snake game added as an early game project on Yichen Hub. It is lightweight, playable in the browser, and useful as a small coding demo.

Score: 0 Running

Project Description

This project demonstrates a basic browser game built with HTML, CSS, and JavaScript. The objective is to control the snake, collect food, and grow longer without crashing into the wall or into the snake’s own body.

Controls

  • Use the arrow keys to move the snake.
  • Eat the red food to increase your score.
  • Avoid hitting the wall or yourself.
